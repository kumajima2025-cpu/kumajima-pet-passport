# 技術架構

## 第一階段
- 前端：HTML / CSS / JavaScript
- 後端：Google Apps Script
- 資料：Google Sheets
- 照片：Google Drive
- 公開網址：Apps Script Web App

## 第二階段
- 前端：React
- Hosting：Firebase Hosting
- 資料庫：Firestore 或 Supabase
- 網域：pet.kumajima.tw
- 內部：admin.kumajima.tw

## API 草案
- GET /pets
- GET /pets/:id
- POST /pets
- POST /records
- GET /passport/:publicCode
- PATCH /pets/:id
